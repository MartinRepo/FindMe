# Findme 🎯

[🇺🇸 English Documentation](README.md)

一个有趣的终端命令行工具，为程序员和技术人员提供每日解压占卜预测。

## 功能特点

- 🎯 **程序员每日解压占卜** - 技术维度分析，确定性每日变化
- 📊 **技术五维评分** - 专注力、创造力、调试手感、协作指数、风险偏好五维度分析
- 🏢 **智能场景检测** - 自动适配工作日（执行/交付）与周末（学习/探索）模式
- 🎲 **确定性生成** - 相同生日+相同日期=一致结果，但每日自动变化
- 🎨 **彩色终端输出** - 美观的彩色界面，带维度可视化进度条
- 💡 **个性化技术建议** - 基于技术维度和当前场景的定制化建议
- 🔬 **开发者体感气压** - 分析本地git/测试/构建数据，提供风险和耐心阈值建议
- 🌍 **多语言支持** - 支持中文和英文两种语言
- 🔐 **生日个性化** - 可选的生日输入，提供个性化分析

## 技术维度

Findme 分析您每日的技术表现，涵盖五个关键维度：

| 维度 | 描述 | 工作日重点 | 周末重点 |
|------|------|-----------|----------|
| 🎯 **专注力** | 专注度和对细节的关注 | 高 (40-100) | 中等 (20-70) |
| 💡 **创造力** | 创新和解决问题的能力 | 中等 (20-80) | 高 (60-100) |
| 🐛 **调试手感** | 故障排除和错误修复技能 | 高 (50-100) | 中等 (30-80) |
| 🤝 **协作指数** | 团队合作和沟通效果 | 高 (60-100) | 低 (10-50) |
| ⚡ **风险偏好** | 尝试新方法的意愿 | 低 (10-60) | 高 (50-100) |

### 场景权重分配

- **工作日模式**: 专注于执行和交付（专注力35%，调试手感30%，协作指数25%，创造力7%，风险偏好3%）
- **周末模式**: 强调学习和探索（创造力40%，风险偏好30%，专注力20%，调试手感7%，协作指数3%）

## 安装

### 从源码编译

```bash
git clone <repository-url>
cd findme
cargo build --release
```

### 安装到系统

```bash
cargo install --path .
```

## 使用方法

### 基本使用

```bash
# 显示今日技术运势（自动检测工作日/周末）
findme

# 显示详细信息和今日运势
findme --verbose

# 个性化分析（输入生日）
findme --birthday "1990-05-15"

# 显示开发者体感气压（分析本地git/测试/构建数据）
findme --pressure

# 组合功能
findme --pressure --birthday "1990-05-15" --language zh

# 指定语言
findme --language zh  # 中文
findme --language en  # 英文

# 设置默认语言
findme --set-language
```

### 语言设置

首次使用时，工具会提示你选择语言。你也可以随时更改语言设置：

```bash
# 设置语言
findme --set-language

# 临时使用不同语言
findme --language en
```

## 技术实现

- **Rust** - 高性能系统编程语言
- **clap** - 命令行参数解析
- **chrono** - 日期时间处理，支持星期检测
- **rand_chacha** - 使用ChaCha20的确定性随机数生成
- **sha2** - SHA256哈希用于种子生成
- **serde** - 序列化框架，为未来模板系统准备
- **tinytemplate** - 动态内容生成的模板引擎
- **colored** - 终端颜色输出，带维度可视化
- **dirs** - 配置文件路径管理

## 运势算法

Findme 使用确定性算法，确保一致性同时提供每日变化：

### 种子生成
```
种子 = SHA256(生日_YYYYMMDD + 今日_YYYYMMDD + 版本 + 盐值)
```

### 确定性特性
- **一致性**: 相同生日 + 相同日期 = 相同结果
- **每日变化**: 每天自动产生不同结果
- **个性化**: 生日输入创建独特模式
- **场景感知**: 自动检测工作日与周末

### 技术维度生成
- 使用 `ChaCha20Rng::seed_from_u64()` 进行确定性随机生成
- 场景偏向范围确保现实的工作模式
- 加权评分适配工作日/周末场景
